# Тестовое задание для компании «Alef»

## Cтек

- Vue
- Pinia

## Развертывание проекта

- Скопируйте репозиторий
  - `https://github.com/eugened503/alef-test.git`
- Установите пакеты
  - `npm install`
- Запустите версию для разработки (ее можно увидеть на локальном сервере)
  - `npm run dev`
